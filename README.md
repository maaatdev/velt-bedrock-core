# VeltBedrockCore

**VeltBedrockCore** was a community-driven Minecraft: Bedrock Edition server core built with [PocketMine-MP](https://github.com/pmmp/PocketMine-MP).
It provided essential features for PvP servers such as duels, custom items, staff utilities, anti-cheat, and world management.

> ⚠️ **Note:** This project is no longer maintained.

## Features

- **Duel System** – Manage player vs. player matches
- **Custom Items** – Extend gameplay with unique items
- **Staff Commands** – Tools for admins and moderators
- **World Management** – Auto-load arenas and control worlds
- **Permanent Night Mode** – Fixed world time for PvP environments
- **Anti-Cheat System** – Basic protection against cheating
- **Database Integration** – Persistent data storage
